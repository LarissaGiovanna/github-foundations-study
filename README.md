A website featuring a complete study guide for the **GitHub Foundations** certification exam.

### Study Modules

The guide is divides into 11 modules:
1. **Introduction to Git and GitHub**: Basic concepts of version control with Git, repositories, commits, branches, and pull requests on GitHub.
2. **GitHub Products**: Main GitHub services, such as GitHub Actions, GitHub Packages, GitHub Security, and others.
3. **Code Scanning and Security**: Security practices and tools on GitHub.
4. **Introduction to GitHub Copilot**: How to use GitHub's AI.
5. **Introduction to GitHub Codespaces**: How to use GitHub's cloud development environments.
6. **Project Management**: Tools to organize and track projects on GitHub.
7. **Introduction to Markdown**: How to format and express yourself through documents on GitHub.
8. **Open Source Projects**: How to contribute to open source projects.
9. **Inner Source Projects**: Applying open source practices within organizations.
10. **Repository Security**: Permission management and repository protection.
11. **GitHub Administration**: Team management and more advanced configurations.

### About the Exam

The **GitHub Foundations** exam is an official GitHub certification that validates fundamental knowledge about: 
- Version control with Git;
- Collaboration on GitHub;
- GitHub products and features;
- Security practices;
- Community development.

#### Financial Information About the Exam
- The exam is **online** and can be taken at your own pace;
- The price may vary depending on location;
- **Students can obtain a free voucher to take the exam by enrolling in [GitHub Education](https://education.github.com)**.
