Fazer a prova de certificação

Quando você se sentir preparado, inscreva-se para o exame de certificação GitHub Foundations. O exame é online e pode ser feito no seu próprio ritmo.

O exame é pago, e o valor pode variar dependendo da sua localização. Porém, ao se inscrever no programa do GitHub Education, o estudante ganha um voucher exclusivo para realizar a prova de forma gratuita.

Por que obter a certificação GitHub Foundations?

Obter a certificação GitHub Foundations demonstra seu conhecimento e habilidades no uso do GitHub, uma das plataformas de desenvolvimento mais populares do mundo. Isso pode abrir portas para novas oportunidades de carreira, aumentar sua credibilidade profissional e ajudá-lo a se destacar no mercado de trabalho.
